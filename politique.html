Politique privative de Nexus Roleplay


Dernière mise à jour : 04/12/2024


Nous prenons votre vie privée très au sérieux. Cette politique de confidentialité vous informe sur les types de données que nous collectons, la manière dont nous les utilisons et les mesures que nous prenons pour les protéger lorsque vous interagissez avec NexusRoleplay sur Discord.

1. Collecte et Stockage de vos données


NexusRoleplay peut recueillir les types d'informations suivants :


De la collecte :

  • Informations sur les serveurs Discord

    • Nous collectons des données relatives aux serveurs où NexusRoleplay est installé, telles que les identifiants de serveur, les noms de serveur, ainsi que les détails sur les canaux, les rôles et les membres.


  • Données d'interaction

    • Nous collectons les informations sur les interactions des utilisateurs avec le bot, y compris les commandes utilisées, les choix effectués dans les scénarios de jeu de rôle, et les réactions aux messages.


  • Données de connexions

    • Nous collectons toutes les informations relatives à l'appareil se connectant comprenant l'adresse ip, la navigateur utilisé et sa version, le nom et type de l'appareil ainsi que le nom du fournisseur d'accès à internet.


Du stockage :

Nous stockons de façon sécurisée tous les types de données précédemment cités dans une base de données externe.

2. Utilisation des données


Les informations que nous collectons via NexusRoleplay sont utilisées pour :


  • Permettre la sauvegarde d'informations

    • Nous utilisons la collecte puis le stockage de données tel que l’argent fictif, les différents rôles de jeu vous étant attribués...


  • Amélioration du service

    • Les données collectées permettent une analyse approfondie de notre équipe permettant d’améliorer la qualité de nos services que l’on peut vous apporter.


  • Support utilisateur

    • Vos données peuvent être utilisées de façon appropriée afin de répondre à vos demandes d’assistance suite à des questions ou d’éventuels problèmes rencontrés.

3. Sécurité des données


Nous mettons en œuvre des mesures de sécurité rigoureuses pour protéger vos informations. Si des données personnelles sont divulguées par inadvertance via NexusRoleplay, nous nous engageons à les supprimer rapidement et à empêcher tout accès non autorisé. Cependant, aucune action ne sera prise si les utilisateurs partagent volontairement leurs informations personnelles.

4. Partage des informations


Les informations collectées via NexusRoleplay ne sont partagées à des tiers extérieurs qu’à l’équipe de gestion du bot NexusRoleplay.

5. Modifications de la politique de confidentialité


Cette politique de confidentialité peut être mise à jour périodiquement. Toute modification sera communiquée via des annonces sur les serveurs Discord associés à NexusRoleplay.

6. Contact


Pour toute question ou préoccupation relative à cette politique de confidentialité, veuillez nous contacter via notre Support Discord.
